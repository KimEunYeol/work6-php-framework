화면 영역 : HTML, CSS, JAVASCRIPT 및 각종 파일들이 위치하고 있습니다.

코어 프로그램 커스터마이징(예제)

controller 내에 파일을 복사하여 custom 경로에 복사/붙여넣기 후 custom에 복사한 파일을 수정해 줍니다.

프로그램(또는 클래스) 추가

1. custom 경로에 원하는 클래스명으로 파일을 추가 합니다. 파일명은 반드시 소문자로 합니다. 
                <?
                    namespace controller\admin;
                    use common;
                    class Sample extends common\AdminLibrary{

                        function __construct(){

                            parent::__construct();

                        }

                        # your code ...
                        function test(){

                        }

                    }
2. 파일명과 동일한 이름으로 클래스를 만들어 줍니다. 클래스명 첫 글자는 반드시 대문자로 합니다.

추가된 DB 테이블 선언

                <?
                    $table['place'] ='w_place';
새롭게 추가된 테이블은 custom/conf/tables.php 파일에 추가 합니다.

DB 테이블 사용법

                <?
                    namespace controller\admin;
                    use common;
                    class Sample extends common\AdminLibrary{

                        function __construct(){

                            parent::__construct();

                        }

                        # your code ...
                        function test(){
                            $query = "select * from ".$this->db->tables['place']." where 1";
                        }

                    }
$this->db->tables['테이블명']으로 불러와 사용할 수 있습니다.

                <?
                    $admin_menu[2][0] = array('디지털광고판','!');

                    $admin_menu[2][1] = array('홍보터관리','/admin/adver/places');
